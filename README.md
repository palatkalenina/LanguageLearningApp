iOS приложение для изучения иностранных языков** на **SwiftUI + SwiftData**. 
Мой первый большой проект с полным стеком iOS разработки.

##  Описание
Приложение с **интерактивными заданиями** для изучения языков:
- Перевод слов
- Составление предложений  
- Multiple choice тесты

**Полный функционал**: авторизация, гостевой режим, выбор языка, 6 уровней, прогресс, локальное хранение.
